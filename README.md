# Gosleap

## Usage

Grace Hopper leaps - hops? - across the screen in a whimsical arc, spreading April Fools' joy everywhere.

## Installation

Add this line to your application's Gemfile:

    gem 'graceleap'

And then execute:

    $ bundle

Or install it yourself:

    $ gem install graceleap
